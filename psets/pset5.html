Problems

Write out answers for the following problems.

  1. Let $F_n$ denote the $n^{th}$ number in the Fibonacci sequence. The Fibonacci sequence is defined recursively as $F_0 = 0$, $F_1 = 1$, and for $n \geq 2$, $F_n = F_{n-1} + F_{n-2}$.

    Let $G_n$ be defined recursively as $G_{-1} = 0$ and $G_0 = 1$ and for $n \geq 1$, $G_n = G_{n-1} + G_{n-2} + 1$. Prove that for all $n \geq -1$, $G_n = F_{n+3} - 1$.

    In your proof, be sure to clearly define a predicate $P(n)$. See the example proof (as well as many other examples in the textbook and from class).

  2. Prove that for all $h \geq 0$, $G_h \geq 2^{h/2}$. Again, you should have a clearly defined predicate $P(n)$. (If you wish, you could name this one $P_2$ to avoid confusion witht he predicate from the first problem.)
  3. A binary tree $T$ is height balanced if for any node in the tree, the height of the left subtree and the height of the right subtree can differ by at most 1. When a tree is height balanced, we can derive a lower bound on the number of nodes. Prove this claim: if binary tree $T$ is height-balanced, $nodes(T) \geq G_{h(T)}$. (In your proof, you are free to use the following fact, if $T$ is height balanced, then any subtree of $T$ is also height balanced.)
  4. Prove that the height balanced criteria is critical in the previous claim. In other words, consider the following claim: for any binary tree $T$, $nodes(T) \geq G_{h(T)}$. Prove that this claim is false.
  5. Prove that a non-empty height balanced tree with $n$ nodes has height at most $2 \log_2 n$. (This is property turns out the useful in practice: for example, if we can design a height balanced binary search tree, then this means any item can be found in at most $2\log n$ steps, which is a small number even when $n$ is large.) Hint: you can show this combining results you’ve already proven.
