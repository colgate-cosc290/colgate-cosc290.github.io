Additional LaTeX Tips

The first problem set has links to numerous LaTeX resources. Since this problem set is on propositional logic, here is a quick cheat sheet for logic symbols:

  • Not: $\lnot$ is \lnot
  • And: $\land$ is \land
  • Or: $\lor$ is \lor
  • Xor: $\oplus$ is \oplus
  • If: $\implies$ is \implies
  • Iff: $\iff$ is \iff
  • Nand: $\uparrow$ is \uparrow
  • Universal quantification: $\forall x: P(x)$ is \forall x: P(x)
  • Existential quantification: $\exists x: P(x)$ is \exists x: P(x)

Problem 3: Circuits

For this question, even though the book asks you to “draw” a circuit, I don’t want you to actually draw a circuit. Instead, I want you to write a propostion. Here’s an example:

  • Problem: The light is on when the inputs are ${ p, q }$ and the light is off when the inputs are ${ p }$ or ${ q }$
  • Answer: an acceptable answer is $p \land q$. Another acceptable answer would be $(p \land q) \lor r$ because the problem didn’t say anything about $r$ and so any circuit consistent with the specified behavior is acceptable. The first answer is preferred because it’s more succinct.
