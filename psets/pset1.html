Instructions

Using LaTeX

Please write up your answers in LaTeX. LaTeX is a mark up language that is particularly powerful and flexible when it comes writing math expressions. (The author of your textbook wrote the textbook using LaTeX and all Computer Science researchers that I know use LaTeX to write their papers. Its math component is also used on sites like StackOverflow, etc.)

You can install LaTeX on your own machine or, even easier, you can use a free online platform like Overleaf or ShareLatex.

Here is a template for writing problem sets. (When the link opens, click “create clone”.)

Non LaTeX submissions: I will also accept work written in some other format (Word doc, by hand, etc), but I will deduct up to 5 points (out of 100) for work submitted in a format other than LaTeX. This is because I want you to learn LaTeX because (a) it’s practically useful, (b) even if you don’t end up using LaTeX again, it will help you develop the ability to quickly learn a new markup language, a skill you will undoubtedly benefit from in the future.

Formatting Guidelines

Please adhere to the following formatting requirements. These will simplify grading so I can more time giving you feedback and less time searching for your answer.

  • Please complete the problems in the order given.
  • A numbered problem should always start at the top of new page. It is okay for your answer to span multiple pages, but when you finish a problem, please start the next problem on a new page.
  • Some numbered problems contain many parts (1.1, 1.2, 1.3, etc.), each part typically being a fairly short answer. Multiple parts can and should appear on the same page.

If you use the provided LaTeX template, these formatting guidelines will be adhered to more or less automatically.

Submitting your work

Upload your answers to Gradescope. (If you hand write any/all of your answers, you will need to scan in each page. Be sure to leave enough time to do this.)

When you upload to gradescope, be sure to follow the Gradescope instructions closely. In particular, you are expected to assign labels to the pages of the uploaded PDF indicating which problems are written on each page.

Problems

Write up answers to the following problems. You can turn them in on pencil and paper.

Problem 1: Warm up

1.1. Sets can be defined via enumeration or abstraction. Let’s define $S$ by enumeration: $S := \{ 1, 2, 3 \}$ and let’s define $T$ by abstraction: $T := \{ (x + 1) \bmod 4: x \in S \}$. What is $T$? Enumerate the elements of $T$.

1.2. Let $A := \{ a, b, c \}$. What is $\mathcal{P}(A)$, the power set of $A$?

1.3. Let $bitstrs := \{ 0, 1 \}^3$. What are the elements of $bitstr$?

Problem 2: Generating power sets through iteration

2.1. Let $A$ and $bitstr$ be defined as above. Show how the elements of $\mathcal{P}(A)$ can be represented by binary sequences of length three. In other words, describe a function $f$ that is a bijective mapping between the elements of $bitstr$ and $\mathcal{P}$. Your answer can simply be a list of $x$, $y$ pairs such that $f(x) = y$. The mapping should make logical sense.

Hint: the all zeros bit string should map to the empty set and the all ones bit string should map to the set $A$. In other words, you should define $f$ such that $f(\langle 0,0,0 \rangle) = \{ \}$ and $f(\langle 1,1,1 \rangle) = \{ a,b,c \}$.

2.2. Generalize your answer to the previous question for any set $S$ of size $n$. Let $S := \{ s_1, s_2, \dots, s_n \}$. Describe a bijective function $f$ between $\{ 0, 1\}^n$ and $\mathcal{P}(S)$. You can use the notation $\langle x_1, x_2, \dots, x_n \rangle$ to refer to an element in $\{ 0, 1 \}^n$ where $x_i$ is either 0 or 1. You can use $s_j$ to refer to the $j^{th}$ element of $S$.

Problem 3: Generating a powerset through recursion

3.1. Suppose $S := \emptyset = \{ \}$. What is $\mathcal{P}(S)$? (Hint: It is not equal to the empty set.)

3.2. Consider $A := \{ a, b, c \}$. What is $\mathcal{P}(A - \{ a \})$? (See if you can observe a relationship between $\mathcal{P}(A)$ and $\mathcal{P}(A - \{ a \})$. The next question asks you to formalize this pattern.)

3.3. Let $S$ denote a set of size at least one and let $x$ an some element in $S$. Let $T := S - \{ x \}$. In other words, $T$ is $S$ after removing $x$. Let $PT$ be a new set which is defined as $PT := \mathcal{P}(T)$, the power set of $T$. Define $\mathcal{P}(S)$ in terms of $PT$ and $x$. You may want to use set operations (union, intersection, difference) and set abstraction (see first question).
